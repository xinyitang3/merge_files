# merge_files

一个轻量级的 Python 脚本，用于合并桌面上（或指定文件夹中）的所有 CSV 文件和 TXT 文件，无需安装任何第三方依赖。

## 功能

- 纵向拼接所有 CSV 文件（保留第一个文件的表头，自动跳过后续文件的表头）
- 按行合并所有 TXT 文件为单个文本文件
- 自动检测文件编码（UTF-8 / GBK），避免中文乱码
- 输出文件编码为 **UTF-8 with BOM**，可直接用 Excel、记事本等软件打开
- 自动排除输出文件本身（`merged.csv` 和 `merged.txt`），防止读写冲突
- 跳过空文件，单个文件错误不影响整体合并

## 使用场景

- 从数据采集系统、导出工具获取到多个结构相同的 CSV 报表，需要汇总分析
- 将多个日志文件（.txt）或文本笔记合并为一个文件
- 日常办公中批量整理散落的表格和文本数据

## 环境要求

- Python 3.x（无需安装任何第三方库，仅使用标准库 `os`、`glob`、`csv`）

## 快速开始

1. **克隆仓库或下载脚本**
   ```bash
   git clone https://github.com/yourusername/desktop-file-merger.git
   cd desktop-file-merger
   ```

2. **修改脚本中的文件夹路径（如需要）**
   脚本默认合并当前用户的桌面文件夹：
   ```python
   folder = r"C:\Users\k\Desktop"   # Windows 示例
   ```
   你也可以改为任意有效路径，或使用 `os.path.expanduser("~/Desktop")` 自动获取桌面路径。

3. **运行脚本**
   ```bash
   python merge_files.py
   ```

4. **查看输出**
   合并后的文件将保存在同一文件夹下：
   - `merged.csv` — 所有 CSV 文件的数据按行合并
   - `merged.txt` — 所有 TXT 文件的内容按原始顺序拼接

## 注意事项

- CSV 合并**要求所有源文件具有相同的列结构**（列名与顺序一致）。若列不完全一致，程序仍会按行拼接，但可能导致数据错位，请提前确认源文件结构统一。
- TXT 合并会直接按文件读取顺序拼接，并在每个文件末尾补全换行，避免内容粘连。如需保留文件原始顺序，脚本已按文件名排序（可自行调整）。
- 首次使用建议**备份原始文件**，或先在小范围文件上测试。
- 如果桌面文件较多，合并过程可能需要一些时间，请耐心等待控制台输出进度。
